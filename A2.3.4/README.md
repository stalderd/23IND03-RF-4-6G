# A2.3.4

VSL, CMI, INRIM, LNE, METAS, PTB, TUBITAK and NPL will select a set of stable attenuation standards for a comparison. VSL will provide a coaxial step attenuator for the frequency range up to 50 GHz and VDI will provide a set of fixed waveguide attenuators or a variable attenuator for the extender-based measurements. NPL, CMI, INRIM, LNE, METAS, PTB and TUBITAK will conduct a comparison by circulating the standards among them. The standards will at lower frequencies also be measured against established attenuation standards by NPL and PTB.
